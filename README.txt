ANTWinV2 is the repository for everyone, who only needs the executable.
ANTWinV2.exe: 32-bit executable for Windows 8 and higher
You can find source code at: https://github.com/hrafnass/ANTv2

Um das Spiel zu starten müssen Sie Ihren Namen und Ihr Geburtsdatum eintragen und danach den Startbutton klicken. Das Kommentarfenster ist optional.
Das Gaming-Window kann nur noch mittels ESC verlassen werden, wobei Sie davor gefragt werden, ob Sie speichern möchten.
In der Taskleiste des Hauptfensters können Sie über Datei/Beenden das Programm schließen.
Mittels Hilfe/Über Reaktionsspiel erhalten Sie Informationen zum Spiel und den Spielregeln.
